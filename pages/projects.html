Here are some of my projects over the past few years! Some of them are past class projects and some are in personal and in development on the side. There are many, many more that are unfinished and languishing on my hard drive somewhere. So it goes.

  • Brite Boards

    This was my senior design project. The goal of the app was to help Speech Language Pathologists, AAC Specalists and other AAC stakeholders spend less time working on creating unique boards from scratch by creating a platform for users to create or upload boards, edit, then publish them for other people to then modify to their needs. I did the systems architecture, as well as the front and backend for the collaborative board editor, written in React and using Y.js to sync everything!

  • Library of Congress Search

    Another one in development. Me and my roomate found a huge (19Gb) XML dataset on the library of congress website listing every single book they have, as of 2016. So we decided to make a TUI search engine for it. We parsed it with a python script, threw it into postgres. It's all indexed, now I just have to make the TUI for it!

  • Simple Sign In

    My final project for my web dev 2 class. Using MERN stack and Socket.io, we made a sign in website that kinda feels like kahoot. I did most of the mongo handling, as well as the socket.io check-in and chat room.

  • NLI Model Analysis

    This was my final project for my NLP class, where I took 3 different roBERTa models, finetuned them for the "Contradictory, My Dear Watson" NLI task, then compared their performance. The final paper that I wrote for it is available in the github repo!

  • This Website!

    Made with html, css and fixi.js. Its my version of code golf, but for a website -- how small can I get this puppy before it's non-functional.
